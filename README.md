# power interactive plot

Позволяет интерактивно менять параметры эксперимента и следить за изменением статистической мощности критерия.

**Веб версия** сайта: https://power-interactive-plot.streamlit.app/

### Зачем? 

Интерактивный график зависимости мощности ($1 - \beta$) от размера выборки ($N$) позволяет легче усвоить связь между этими величинами. Кроме того, можно изменять и уровень значимости ($\alpha$), и относительное изменение самой метрики.
