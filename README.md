# power interactive plot

Позволяет интерактивно менять параметры эксперимента и следить за изменением статистической мощности критерия.

### Зачем? 

Интерактивный график зависимости мощности от размера выборки позволяет легче усвоить связь между этими величинами. Кроме того, можно изменять и уровень значимости, и относительное изменение самой метрики.
